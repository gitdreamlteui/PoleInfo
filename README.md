# PoleInfo

## API
### Lancer l'API de test
1. Créer un environnement virtuel python
```py
python3 -m venv env
```

2. Activer l'envrionnement virtuel
```py
source env/bin/activate
```

3. Installer les dépendances 
```py
pip install "uvicorn[standard]"
pip install fastapi
pip install passlib
```

4. Lancer l'API
```py
uvicorn api:app --reload
```
